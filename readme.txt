== Installation ==

### Automatic Installation (Recommended)

1. Log in to your WordPress admin panel
2. Navigate to **Plugins > Add New**
3. Search for "AI Product Review Generator for WooCommerce"
4. Click **Install Now** and then **Activate**
5. Go to **Products > AI Review Generator** to configure

### Manual Installation

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to **Plugins > Add New > Upload Plugin**
4. Choose the ZIP file and click **Install Now**
5. Activate the plugin through the 'Plugins' menu

### Configuration

1. After activation, go to **Products > AI Review Generator**
2. Click on the **Settings** tab
3. Enter your OpenAI API key ([Get it here](https://platform.openai.com/api-keys))
4. Configure your preferences:
   - Select AI model (GPT-4o Mini is FREE)
   - Set reviews per product
   - Choose sentiment balance
   - Configure date ranges
5. Save settings and start generating!

### Getting Your OpenAI API Key

1. Visit [OpenAI Platform](https://platform.openai.com/signup)
2. Create a free account or log in
3. Navigate to API Keys section
4. Click "Create new secret key"
5. Copy the key (starts with 'sk-')
6. Paste it in the plugin settings

== Frequently Asked Questions ==

= Is this plugin free to use? =

The plugin itself is free. You'll need an OpenAI API key, but OpenAI offers free credits for new users, and the GPT-4o Mini model is extremely affordable (almost free for moderate use).

= How much does it cost to generate reviews? =

With GPT-4o Mini (recommended):
- Approximately $0.0001 per review
- 1000 reviews ≈ $0.10
- New OpenAI accounts get free credits

= Are the generated reviews legal to use? =

The plugin is designed for legitimate business purposes such as seeding new products with initial feedback and testing. Always comply with your local consumer protection laws and platform policies. Consider disclosing AI-generated content where required.

= Can Google detect AI-generated reviews? =

Our plugin generates diverse, contextual reviews using advanced AI. However, we recommend using generated reviews responsibly and in combination with genuine customer feedback.

= How many reviews can I generate at once? =

You can generate reviews for unlimited products in a single batch. The plugin uses background processing to handle large batches without timeout issues.

= Can I customize the review content? =

Yes! You can:
- Set custom prompts with product placeholders
- Control sentiment ratios
- Specify focus keywords
- Choose review lengths
- Set custom date ranges

= Will this slow down my website? =

No. The plugin uses WooCommerce's Action Scheduler for background processing, ensuring your site remains fast and responsive during generation.

= Can I delete AI-generated reviews? =

Yes, you can delete individual reviews or all AI-generated reviews with one click from the Reviews management page.

= What languages are supported? =

You can generate reviews in any language supported by OpenAI. Simply specify the language in your custom prompt.

= Is my data safe? =

Yes. Only product information (name, description, price) is sent to OpenAI for review generation. No customer or sensitive data is ever transmitted.

= Do you offer support? =

Yes! We provide support through the WordPress.org forums and comprehensive documentation in the Help section of the plugin.

= Can I use this with other review plugins? =

Yes, the plugin works with standard WooCommerce reviews and is compatible with most review enhancement plugins.
