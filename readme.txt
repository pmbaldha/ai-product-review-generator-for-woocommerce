Transform your WooCommerce store with AI-powered product reviews. Generate authentic, detailed reviews using advanced AI technology.

== Description ==

Transform your WooCommerce store into a hub of engaging customer feedback with AI Product Review Generator. This powerful tool uses advanced AI to create realistic, detailed reviews that help seed new products with authentic feedback and build social proof, resulting in increased customer trust and higher conversions.

= Key Features =

* **Effortless AI-Driven Review Generation** - Automate the creation of natural-sounding reviews for individual products or entire categories
* **Fully Customizable Review Settings** - Tailor review lengths and set custom review prompts using dynamic placeholders
* **Custom Focus Areas** - Direct the AI to highlight specific product features
* **Advanced Sentiment & Tone Control** - Choose from negative, neutral, positive, or mixed sentiments
* **Robust WooCommerce Integration** - Seamlessly integrates with WooCommerce, supporting all core product types
* **Flexible Date Range** - Set custom date ranges for review timestamps
* **Bulk Generation** - Generate multiple reviews for multiple products at once
* **OpenAI Integration** - Powered by OpenAI's latest GPT models

= How It Works =

1. Install and activate the plugin
2. Configure your OpenAI API key in the settings
3. Select products or categories for review generation
4. Customize review settings (length, sentiment, focus areas)
5. Click generate and watch your products gain authentic reviews

= Requirements =

* WordPress 6.7 or higher
* WooCommerce 9.3 or higher
* PHP 7.2 or higher
* OpenAI API key (obtain from OpenAI platform)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-product-review-generator` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Settings > AI Product Reviews to configure the plugin
4. Enter your OpenAI API key
5. Configure your review generation settings
6. Start generating reviews!

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, you need an OpenAI API key to use this plugin. You can obtain one from the OpenAI platform.

= Can I customize the review content? =

Yes, you can set custom prompts, focus areas, review lengths, and sentiments to control the generated content.

= Are the generated reviews marked in any way? =

Yes, generated reviews are marked with metadata for identification, but this is not visible to customers.

= Can I generate reviews for specific products only? =

Yes, you can select specific products or entire categories for review generation.

= What languages are supported? =

You can generate reviews in any language by specifying it in your custom prompt.
